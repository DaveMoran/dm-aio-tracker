# iOS Native Project

This directory is generated by Capacitor on a Mac with Xcode installed.

## First-time setup (run on your Mac)

```bash
# 1. Install dependencies
npm install

# 2. Build the web app and sync to native
npm run cap:build

# 3. Add the iOS platform (creates this ios/ folder)
npx cap add ios

# 4. Open in Xcode
npm run cap:open:ios
```

## Day-to-day development

```bash
# Rebuild web + sync native assets
npm run cap:build

# Open Xcode (after cap:build)
npm run cap:open:ios
```

## iPhone 14 Pro specifics

- Bundle ID: `com.davemoran.dmtracker`
- Deployment target: iOS 16+
- Orientation: Portrait only (set in Xcode → General → Deployment Info)
- Status bar: Dark style, #F7F3EE background
- Safe areas: handled via `env(safe-area-inset-*)` in CSS

## Live reload during development

To use live reload from your local machine, uncomment the `server.url` block
in `capacitor.config.ts` and replace `YOUR_LOCAL_IP` with your Mac's local IP:

```typescript
server: {
  url: 'http://192.168.x.x:5173',
  cleartext: true,
},
```

Then run `npm run dev` on your Mac and rebuild the native app.
